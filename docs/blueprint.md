# **App Name**: Crypto Ice Access

## Core Features:

- Key Entry: Allow users to enter their key for access.
- Access Validation: Validate the entered key against a backend system to grant access.
- Login Button: Provides a login button after the user has inputted their access key.
- Purchase Button: Navigates the user to a purchase flow.
- Contact Support Link: Directs users to a support channel for assistance.
- Particle Animation: Dynamically rendered falling snow particles.

## Style Guidelines:

- Background color: Dark navy/space (#0A192F) to create a futuristic feel.
- Primary color: Electric purple (#BE52FF) for buttons and accents, conveying a tech vibe.
- Accent color: Light blue (#52BFFE) to complement the primary color and add a neon glow effect.
- Font: 'Orbitron', sans-serif, for titles and subtitles to emulate a futuristic pixel/tech style. Note: currently only Google Fonts are supported.
- Centered content container with rounded input field and buttons for a modern, focused look.
- Falling snow particle animation in the background to add visual interest and a sense of immersion.
- Hover animations for buttons (brightness increase + slight scale) to provide interactive feedback.
